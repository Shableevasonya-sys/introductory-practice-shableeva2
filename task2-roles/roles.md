# Задание 2. Роли пользователей платформы

Платформа: **Авито**.

| Роль | Цель на платформе | Функции (что делает) | Результат |
|---|---|---|---|
| Покупатель | Найти и купить подходящий товар или услугу | 1. Ищет объявление по ключевым словам / 2. Фильтрует предложения по цене / 3. Сравнивает похожие товары / 4. Просматривает фотографии товара / 5. Изучает описание объявления / 6. Проверяет рейтинг продавца / 7. Отправляет сообщение продавцу / 8. Оформляет доставку товара | Покупатель находит нужное предложение и совершает покупку на удобных условиях |
| Частный продавец | Продать личные вещи, товар или имущество | 1. Создает карточку объявления / 2. Загружает фотографии товара / 3. Указывает цену товара / 4. Заполняет описание объявления / 5. Выбирает категорию размещения / 6. Отвечает на сообщения покупателей / 7. Редактирует условия продажи / 8. Передает товар покупателю | Продавец получает покупателя и продает товар |
| Профессиональный продавец | Использовать Авито как постоянный канал продаж | 1. Размещает несколько объявлений / 2. Обновляет ассортимент товаров / 3. Настраивает платное продвижение / 4. Анализирует просмотры объявлений / 5. Обрабатывает входящие заявки / 6. Консультирует клиентов в сообщениях / 7. Управляет ценами товаров / 8. Поддерживает репутацию профиля | Компания или предприниматель получает поток клиентов и продажи |
| Исполнитель услуг | Найти клиентов для выполнения работ или услуг | 1. Публикует описание услуги / 2. Указывает стоимость работы / 3. Добавляет примеры выполненных работ / 4. Принимает заявки клиентов / 5. Уточняет детали заказа / 6. Согласовывает время выполнения / 7. Выполняет услугу клиенту / 8. Получает отзыв заказчика | Исполнитель получает заказы и развивает клиентскую базу |
| Работодатель | Найти сотрудников на открытые вакансии | 1. Создает текст вакансии / 2. Указывает должность сотрудника / 3. Описывает обязанности кандидата / 4. Публикует условия работы / 5. Просматривает отклики соискателей / 6. Отвечает кандидатам в сообщениях / 7. Отбирает подходящие резюме / 8. Приглашает кандидата на собеседование | Работодатель получает отклики и находит подходящего сотрудника |
| Соискатель | Найти подходящую работу или подработку | 1. Ищет вакансии по профессии / 2. Фильтрует вакансии по городу / 3. Сравнивает уровень зарплаты / 4. Изучает требования работодателя / 5. Сохраняет интересные вакансии / 6. Отправляет отклик работодателю / 7. Уточняет условия работы / 8. Договаривается о собеседовании | Соискатель находит подходящую вакансию и выходит на контакт с работодателем |
